Venta de Productos

Registro de salidas de inventario por ventas para mantener un control preciso del stock y optimizar la gestión de inventarios.

Instalación de Productos

Registro de instalaciones de productos de videovigilancia para mantener un seguimiento detallado de cada instalación realizada.

Otro tipo de Salidas

Registro de salidas de inventario para cualquier otro tipo de transacción o uso fuera de ventas o instalaciones específicas.
